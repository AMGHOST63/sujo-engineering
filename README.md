## 🚀 How to Publish Your Website for FREE (Step by Step)

### Step 1 — Create a GitHub Account

1. Go to [https://github.com](https://github.com)
2. Click **Sign up**
3. Enter your email, create a password, and choose a username
   - Suggestion: `sujo-engineering` or `jonathansumaili`
4. Verify your email address

---

### Step 2 — Create a New Repository

1. Once logged in, click the **+** button at the top right
2. Select **New repository**
3. Fill in the details:
   - **Repository name:** `sujo-engineering` (exactly like this, no spaces)
   - **Description:** SUJO Engineering Consulting website
   - **Visibility:** Select **Public**
   - Check the box **Add a README file**
4. Click **Create repository**

---

### Step 3 — Upload Your Website File

1. Inside your new repository, click **Add file**
2. Select **Upload files**
3. Drag and drop your `index.html` file into the upload area
4. Scroll down and click **Commit changes**
5. Wait a few seconds for the upload to complete

---

### Step 4 — Enable GitHub Pages (Make It Live)

1. Inside your repository, click **Settings** (top menu)
2. In the left sidebar, scroll down and click **Pages**
3. Under **Branch**, select **main** from the dropdown
4. Leave the folder as `/ (root)`
5. Click **Save**
6. Wait 2–3 minutes

---

### Step 5 — Visit Your Live Website

After a few minutes, your website will be live at:

```
https://[your-username].github.io/sujo-engineering/
```

For example:
```
https://sujo-engineering.github.io/sujo-engineering/
```

🎉 **Your website is now live and free forever!**

## ✏️ How to Update Your Website

Whenever you want to change something on the website:

1. Open `index.html` on your computer with any text editor
   - Windows: Notepad or WordPad
   - Mac: TextEdit
   - Better option (free): Download **VS Code** from [https://code.visualstudio.com](https://code.visualstudio.com)
2. Make your changes (phone number, email, prices, etc.)
3. Save the file
4. Go back to your GitHub repository
5. Click **Add file → Upload files**
6. Upload the updated `index.html`
7. Click **Commit changes**
8. Your website updates automatically within 1–2 minutes

## 🌐 Connect a Custom Domain (Optional)

Once you buy a domain like `sujo-engineering.com`, you can connect it to your GitHub website so it shows your real domain instead of the github.io address. This is free to set up.

### Steps to connect your domain:

1. Buy your domain on [Namecheap.com](https://namecheap.com) (~$12/year)
2. In your GitHub repository, go to **Settings → Pages**
3. Under **Custom domain**, type your domain (e.g. `sujo-engineering.com`)
4. Click **Save**
5. Go to Namecheap, open your domain's **DNS settings**
6. Add these DNS records:

| Type | Host | Value |
|------|------|-------|
| A | @ | 185.199.108.153 |
| A | @ | 185.199.109.153 |
| A | @ | 185.199.110.153 |
| A | @ | 185.199.111.153 |
| CNAME | www | [your-username].github.io |

7. Wait 24–48 hours for the domain to connect
8. Your website will then be live at `www.sujo-engineering.com`
